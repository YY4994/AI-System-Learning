# AI System Learning Path

> 北京理工大学软件工程硕士 | 聚焦AI系统/软件研发方向

这是一个记录我个人学习AI系统开发（深度学习框架、高性能计算、模型推理优化）的完整项目仓库。我通过动手实现核心模块和参与开源来构建扎实的工程能力。

## 🎯 学习目标

- **深入理解** PyTorch 等现代深度学习框架的内部机制。
- **掌握** CUDA 编程与 GPU 高性能计算的核心技能。
- **具备**从零构建AI系统核心组件（如自动求导、推理引擎）的能力。
- **最终目标**：获得头部科技公司或央企的AI系统研发（实习）岗位。

## 📂 项目导航

以下项目按学习路径顺序排列，难度递增。

### 阶段一：内功筑基 - PyTorch与CUDA核心

- **[项目1: 自定义CUDA算子扩展](projects/01_cuda_swish_extension/)** - 为PyTorch实现一个高性能的Swish激活函数CUDA算子，并进行性能优化分析。
- *目标：掌握框架扩展与底层性能优化。*

### 阶段二：系统实战 - 从零构建框架

- **[项目2: 微型深度学习框架](projects/02_tiny_dl_framework/)** - 仅使用C++/CUDA，实现包含张量、计算图自动求导、优化器的轻量级训练框架，并在MNIST上验证。
- *目标：系统性理解深度学习框架的全貌。*

### 阶段三：生态参与 - 工业级实践

- **[项目3: 开源项目贡献](#)** - 计划为PyTorch或ONNX Runtime等开源项目贡献代码。
- *目标：体验大型工程协作，解决真实问题。*
